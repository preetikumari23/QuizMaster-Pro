# QuizMaster Pro 🎯

A responsive and interactive quiz application built using **HTML, CSS, and JavaScript**. The project provides an engaging quiz experience with dynamic question handling, score tracking, and a modern user interface.

## 🌐 Live Demo

https://preetikumari23.github.io/QuizMaster-Pro/

## 🚀 Features

* Interactive multiple-choice questions
* Real-time score tracking
* Instant answer feedback
* Responsive design
* Clean and modern UI
* Dynamic quiz functionality

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
